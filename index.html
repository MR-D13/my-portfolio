À propos

Qui suis-je ?

Je suis passionné par l’informatique sous tous ses aspects (hardware, software & network).
Principalement pour le côté Hardware car le fonctionnement de la machine m'a toujours fasciné.
En 2020, je suis devenu intégrateur web en autodidacte, pour faire connaissance avec le domaine du web.
Cette année, je souhaite vraiment travailler dans l'informatique pour devenir technicien informatique.

Pourquoi ce portfolio ?

Ce portfolio affiche mes créations ainsi que mes compétences.
J'améliore quotidiennement mes connaissances dans ces 3 domaines : Hardware, Network et Software.

Actuellement :
• faire connaissance avec l'environnement Linux
• tester VirtualBox sous différents systèmes d'exploitations
• apprendre la simulation des topologies de réseaux avec GNS3

• si j'ai le temps, bien apprendre PHP/MySQL
